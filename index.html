Алгоритмы

Условия

        Количество кофе 125 г
         
        Кофе более или равно 16 г?
          Приготовить капучино
         
        Иначе
          Сходить в магазин и купить кофе
         
      
        const coffeeWeight = 125;
         
        if (coffeeWeight >= 16) {                    
          makeCappuccino();
        }
        else {
          buyCoffee();
        }

Условия

        Посетитель авторизовался
         
        Посетитель авторизован?
          Показать аватар посетителя
         
        Иначе
          Показать кнопку авторизации
         
      
        const userAuthorized = true;                    
         
        if (userAuthorized) {
          showAvatar();
        }
        else {
          showLoginButton();
        }

Условия

        У покупателя в корзине товаров на 2400 р
         
        Посетитель покупает на 2500 и более?
          Подсчитать скидку в 10%
         
        Иначе, если покупает на 2000 и более, но менее чем на 2500
           Подсчитать скидку в 6%
         
        Иначе
           Подсчитать скидку в 2%
         
      
        const price = 2400;                    
         
        if (price >= 2500) {
          getDiscount(10);
        }
        else if (price >= 2000 && price < 2500) {                        
          getDiscount(6);
        }
        else {
          getDiscount(2);
        }

Javascript

Javascript — прототипно-ориентированный однопоточный асинхронный скриптовой язык программирования.

  • Прототипно-ориентированный — почти у всего есть прототип.
  • Однопоточный — одна инструкция в единицу времени.
  • Асинхронный — код может выполниться не сразу.
  • Скриптовой — нуждается в интерпретаторе (браузер, node.js).
